Group of commands to manage and inspect clusters.
See `replictl cluster --help` for general information.

## refresh
Schedule a cluster refresh task.

```text
$ replictl cluster --cluster <CLUSTER> --namespace <NAMESPACE> refresh [OPTIONS]
```
