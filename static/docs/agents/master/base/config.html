Configuration

Configuration

As agents are delveloped as independent tools, configuration options (and sometimes format) differ from agent to agent.

Official agents and those implemented using the base rust crate provide a set of standard configuration options:

# Datastore independent agent configuration.
agent:
  # HTTP server configuration.
  server:
    # Address and port to bind the server to.
    bind: '127.0.0.1:8000'

  # Distributed tracing configuration.
  tracer:
    # Distributed tracing backend to use.
    #
    # Avaliable options:
    #
    #   - NoopTracer
    #   - ZipkinTracer
    backend: 'NoopTracer'

    # Zipkin specific tracer configuration.
    #zipkin:
    #  service_name: 'agent'
    #  topic: 'zipkin'
    #  kafka:
    #    - 'localhost:9092'
